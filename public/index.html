Allgit

Get files from Git services, with correct headers

Allgit is a service that can serve files from Github and Gitlab repositories, but with the correct Content-Type headers. It's entirely free, forever.

For Github:

https://allgit.rk1.us/gh/user/repo@branch/file_path_here
-- Get a file from a branch

For Gitlab:

https://allgit.rk1.us/gl/user/repo@branch/file_path_here
-- Get a file from a branch
